PostHub is a blogging web application developed using the Django framework. It allows users to publish their thoughts and articles in the form of blog posts. The main purpose of this project is to understand CRUD operations, URL routing, template rendering, and database management in Django through a practical example.

# Features
- Create new blog posts
- Edit and update existing posts
- Delete posts
- View all blog posts
- View individual post details

# Tech Stack
* Backend: Django, Python
* Frontend: HTML, CSS, Bootstrap

# Database:
SQLite (default Django DB)

# Purpose
This project helps beginners learn and practice core Django concepts while building a fully functional blog system.
